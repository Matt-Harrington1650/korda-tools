# AI + Storage Control Matrix (v1)

| Risk | Prevention | Detection | Logging/Audit Evidence | Severity (H/M/L) | Owner | Test Procedure |
|---|---|---|---|---|---|---|
| Cross-client leakage | Enforce project/workspace boundary at service layer; mandatory ProjectContext token on every query | Boundary-violation detector checks response citations for foreign project ids | `audit_log` events for request scope, denied joins, and response citations | H | Security Architect | Run query in Project A referencing Project B artifact id; confirm hard deny and audit event with `scope_violation`. |
| Prompt injection via documents | Strip/neutralize instruction-like payloads during ingestion; model guardrail policy ignores document-executed instructions | Injection signature scanner flags adversarial phrases and tool-call attempts | Ingestion scan result + policy decision + blocked execution trace | H | AI Platform Lead | Upload document containing "ignore all prior instructions"; verify scanner flag and policy block before response generation. |
| Outdated precedent surfacing | Retrieval filter enforces latest-issued and non-superseded sources by default | Response validator checks if cited artifact status is `Superseded`/`Archived` | Citation status snapshot and resolver decision in audit chain | H | Knowledge Governance Lead | Seed one superseded and one current spec; ask precedent question; verify current cited, superseded excluded unless explicit override. |
| Ingestion drift (silent coverage drop) | Coverage SLA thresholds and required feed health checks before daily close | Drift monitor compares expected vs actual ingested artifact counts per project | Daily coverage report + alert events + acknowledgment trail | H | Data Operations Lead | Disable one ingestion source for 24h; verify alert fires when coverage delta exceeds threshold and incident logged. |
| Metadata drift/corruption | Schema validation + strict enum checks + signed migration pipeline | Drift checker compares current metadata distribution against baseline profile | Migration/apply logs, checksum mismatches, rollback records | H | Data Platform Lead | Inject invalid enum/status values; verify reject, no commit, and data-quality event logged. |
| PII exposure in transcripts/emails | PII classifier + redaction before persistence and external AI calls | DLP scan on outgoing prompts and stored transcripts | Redaction diff log, DLP findings, outbound block/allow decisions | H | Privacy Officer | Upload email with SSN/phone; verify redaction in stored transcript and deny external AI export without override. |
| Superseded drawings cited in answers | Citation resolver constrained to `Issued/Record/AsBuilt` states unless reviewer override | Post-answer citation compliance check detects superseded references | Response compliance event with cited artifact status map | H | Design QA Lead | Ask question that matches superseded sheet text; verify response omits superseded citation and records compliance pass. |
| Broken retention policy execution | Retention engine enforces minimum years + legal hold precedence | Retention auditor scans delete queue against policy constraints | Delete job ledger + hold checks + denied-delete evidence | H | Records Manager | Attempt deletion of artifact under legal hold; confirm deletion blocked and hold enforcement logged. |
| Export abuse (bulk exfiltration) | Role-based export limits, watermarking, and throttled bulk export caps | Anomaly detector flags unusual export volume/patterns | Export audit with actor, count, classification, destination | H | Security Operations | Execute scripted export burst above threshold; verify throttle/deny and alert with user/session evidence. |
| Unapproved external AI usage | Default-deny provider policy; allowlist required per classification | Outbound gateway checks provider id against approved list | Outbound request log with provider, policy rule id, decision | H | Security Architect | Configure non-allowlisted model endpoint; verify outbound blocked and policy violation audit captured. |
| Incomplete citation chain | Response contract requires citations for non-trivial claims | Response linter fails answers with missing/invalid citations | Lint failure logs, response rejection record, retry trace | M | AI Platform Lead | Force model answer without citations in test harness; verify response rejected and retry or fail-safe notice returned. |
| Conflicting source interpretation | Conflict detector compares cited artifacts by revision/date/status | Conflict score triggers human-review flag | Conflict detection event including artifact ids and rule results | M | Domain QA Lead | Provide two issued docs with conflicting values; verify conflict flag and mandatory review state. |
| Hallucinated code/spec references | Retrieval-only answer mode for regulated prompts; forbid uncited assertions | Uncited-claim detector scans response spans for unsupported references | Span-level provenance map and uncited-claim violations | H | AI Safety Lead | Ask for clause not present in corpus; verify system states "not found" and produces no fabricated citation. |
| Vector index stale against authoritative corpus | Scheduled re-index on artifact state change; embedding job retries with DLQ | Freshness monitor compares vector timestamp vs artifact timestamp | Re-index job logs, lag metrics, stale-index alerts | M | Search Platform Lead | Update artifact revision and skip index job; verify stale-lag alert and query fallback to metadata search. |
| Unauthorized manual metadata edits | RBAC for metadata override and mandatory reason field | Override monitor flags high-risk field edits (status/classification) | Override audit event with before/after, actor, reason | M | Data Steward | Attempt metadata status override without role; verify deny and no DB write. |
| Misclassification of sensitivity level | Required classification at ingest + classifier assist + reviewer confirmation for sensitive classes | Periodic sampling and high-risk misclassification detector | Classification decision log + reviewer approvals + correction history | H | Privacy Officer | Ingest client-confidential sample labeled public; verify classifier challenge and forced reviewer confirmation. |
| Checksum mismatch between object and metadata | Compute/verify sha256 at ingest and read path | Integrity verifier periodically re-hashes sampled objects | Integrity scan reports and mismatch incident entries | H | Storage Reliability Lead | Alter stored blob bytes in test fixture; verify integrity scan failure and quarantine workflow. |
| Backup/restore inconsistency | Point-in-time backup with checksum manifest and restore verification job | Restore drill compares record counts, hash manifests, and chain validity | Backup manifest, restore report, chain verification evidence | M | Infrastructure Lead | Perform restore drill from prior backup; verify counts/hash chain match and report archived. |
| OCR extraction failure on low-quality scans | Minimum scan quality gate and manual OCR queue fallback | OCR confidence threshold monitor routes low-confidence docs to review | OCR confidence logs, queue entries, reviewer outcomes | M | Document Processing Lead | Upload blurred scanned sheet; verify low-confidence routing and no auto-authoritative extraction. |
| Credential leakage in prompts/logs | Secret detector on prompts/logs + vault-only secret handling | Secret scanning in CI and runtime log pipeline | Secret scan findings and redaction/remediation audit trail | H | Security Operations | Inject fake API key string in prompt content; verify redaction and blocked persistence of raw token. |

- Matrix contains at least 10 additional engineering-firm risks beyond mandatory list.
- Every row includes concrete prevention, detection, evidence, owner, and explicit test procedure.
