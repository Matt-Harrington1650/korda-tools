# Quick Start

## First boot checklist

1. Open **Settings** and verify defaults.
2. Decide whether **Developer Mode** should be enabled for this install.
3. Add at least one tool in the registry or import a custom tool package.
4. Open **Workflows**, create a minimal workflow, and run a test.
5. Confirm logs and outputs are visible.

## Minimal setup

| Step | Action | Where |
|---|---|---|
| 1 | Add tool config | Registry pages |
| 2 | Save credentials (if needed) | Settings / tool forms |
| 3 | Add or import a custom package | Tools Library |
| 4 | Create a workflow | Workflows page |
| 5 | Run and validate | Workflows or Chat |

## Useful shortcuts

```text
Dashboard   -> quick status and tool filtering
Workflows   -> orchestrated runs and schedule setup
Tools       -> custom package create/import/export
Help Center -> docs, runbooks, troubleshooting
```
