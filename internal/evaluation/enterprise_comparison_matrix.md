## Findings (severity-ranked)
- Critical
  - Project-boundary bypass exists in finalize/integrity paths when artifact IDs are known.
- High
  - UI/tool execution path performs direct outbound model/API calls from frontend execution pipeline; enterprise policy mediation is not centralized server-side.
  - Classification and AI confidence/citation policy controls are largely docs/schema-only, not runtime-enforced.
  - TS migration runner remains integration hook (`openSqlite` TODO), reducing confidence in non-Tauri startup migration authority.
- Medium
  - Release tagging implementation mismatches governance convention.
  - CODEOWNERS remains placeholder and weakens operational ownership/accountability.
  - Runbooks exist but restore/audit continuity drills are not evidenced in CI or automated validation.
- Low
  - Lint warnings indicate non-blocking React compiler compatibility issues.
